What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating web pages
  • HTML describes the structure or blueprint of a web page
  • HTML elements tell the browser how to display the content

What Elements Make up a Webpage?

  1. The <!DOCTYPE HTML> declares that this is a HTML5 document.
  2. <HTML> element is the root element of an HTML page.
  3. <TITLE> elements specifies a title for the HTML page(Shown in the browsers title bar or tab)
  4. <BODY> element defines the documents body, and is a container for all the visible contents, such as
  5. <h1> element defines a large heading
  6. <p> element defines a paragraph
